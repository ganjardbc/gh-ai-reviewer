# Blocked Tasks Register

This register catalog details tasks that have failed validation, dependency checks, or are paused awaiting human developer intervention:

-   **Task ID**: Task [Phase].[Number]
-   **Blocked Date**: [YYYY-MM-DD]
-   **Blocking Reason**: [Unresolved compiler error / credentials config missing / merge conflict]
-   **Required Action**: [Step description to clear the blocker]
-   **Owner / Assignee**: [Human developer / System Administrator]
-   **Blocking Dependency**: Task [Number]
